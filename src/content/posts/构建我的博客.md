# 记录构建博客的过程

`Fork（克隆）` 博客仓库 [fuwari](https://github.com/Besty0728/fuwari) 到我的账号中。

通过这篇文章来记录构建博客的过程，先创建一个文章：

```bash
pnpm run new-post -- "文章标题"
```

## 博客文章结构

```bash
---
title: 文章标题                    # 必填
published: 2025-09-02T20:10:14    # 发布时间
description: '文章描述'            # SEO描述
image: 'https://example.com/img.jpg'  # 封面图片
tags: [标签1, 标签2, 标签3]       # 标签
category: 分类名称                # 分类
draft: false                      # 是否为草稿
lang: ''                         # 语言(默认继承配置)
pinned: true                     # 置顶文章（显示在列表最前面）
prerenderAll: true               # 预渲染所有内容（适用于长文章，确保滚动流畅）
---
```

> **置顶文章**：添加 `pinned: true` 即可将文章置顶显示；**长文章优化**：添加 `prerenderAll: true` 可预渲染折叠区内容，确保滚动流畅。

在 [夜轻 Blog](https://blog.yeqing.net/acg-api/) 中为这篇文章添加封面图。

## 为自已的博客绑定域名

> [免费域名 + Cloudflare 托管｜告别 ClouDNS | 2026 亲测可用](https://www.bilibili.com/video/BV17jcJzTEn1/?spm_id_from=333.1007.top_right_bar_window_custom_collection.content.click&vd_source=8abceb502969e7de8c2eb9bc66a1d6e3)、[Cloudflare托管域名，免费CDN加速，免费申请有效期15年的证书](https://zhuanlan.zhihu.com/p/650354462)

在 [GNAME](https://gname.vip/) 中 **免费注册** 一个 `.eu.cc` 二级域名。在注册过程中，会赠送**三张优惠卷**，这样可以白嫖三年这个域名。

![](/images/myblog/Snipaste_2026-03-09_19-48-51.png)

点击 [cloudflare](https://dash.cloudflare.com/1910d672d34b4e644620b5e437750c60/add-site) 开始绑定白嫖到的域名：

![](/images/myblog/Snipaste_2026-03-09_20-23-05.png)

计划选择免费版，点继续：

![](/images/myblog/Snipaste_2026-03-09_20-24-43.png)

这里 Cloudflare 会扫描域名原有的 DNS记录 并复制过来 如果你是新购买的域名，可能会有一些默认的解析记录，可以全部删掉，然后点击继续前往激活：

![](/images/myblog/Snipaste_2026-03-09_20-27-38.png)

提示是否稍后添加记录，点 Confirm 确认：

![](/images/myblog/Snipaste_2026-03-09_20-28-42.png)

这里就可以看到 Cloudflare 提供的 NameServer 了，接下来需要到你的 DNS提供商（如果是新注册的域名，那就到你的域名注册商）的网站中进行操作：

![](/images/myblog/Snipaste_2026-03-09_20-30-05.png)

接下来就是在 [GNAME](https://gname.vip/user#/domain_info/ym=amongruins.eu.cc) 白嫖的域名中修改 `域名DNS` 了

![](/images/myblog/Snipaste_2026-03-09_20-33-32.png)

域名注册商可能需要 24小时 来处理 nameserver 的更新，当你的网站在 Cloudflare 上激活后，你会收到一封邮件通知：

![](https://pic1.zhimg.com/v2-99ef5caefdecb5c3f53ff2146a098790_1440w.jpg)

回到首页，也可以看到已添加的域名当前的状态。一般来说，几分钟状态就激活了：

![](/images/myblog/Snipaste_2026-03-09_20-42-28.png)

## 托管博客

在 CloudFlare 的 `Workers & Pages` 中 [Create application](https://dash.cloudflare.com/1910d672d34b4e644620b5e437750c60/workers-and-pages/create) 连接 Github 的博客仓库：

![](/images/myblog/Snipaste_2026-03-09_20-48-34.png)

选择自已的博客仓库，`Next`（下一步）后开始部署：

![](/images/myblog/Snipaste_2026-03-09_20-52-31.png)

部署完成后会到达此界面，并添加绑定的白嫖域名：

![](/images/myblog/Snipaste_2026-03-09_20-58-10.png)

连接白嫖域名：

![](/images/myblog/Snipaste_2026-03-09_20-59-22.png)
